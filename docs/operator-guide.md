# Operator Guide

## First Deployment
1. Register operator, missions, credentials, personnel and robots.
2. Import existing dispatch and reporting history.
3. Run read-only credential-scope and robot mission dry-runs.
4. Configure safety-class allowed sets and human sign-off paths.
5. Publish a dry-run report and audit export.

## Minimum Production Controls
- credential scope validation before any dispatch
- governor gate on every robot action before dispatch
- human sign-off for :high/:safety-critical robot actions
- evidence-backed public reports
- audit export for every dispatch, sign-off and report
- backup manual dispatch process

## Certification
Certified operators must prove robot-safety integrity, credential-scope
discipline, evidence-backed reporting and human review for dispatch-affecting
actions.
